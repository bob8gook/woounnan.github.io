# 환경구성

### sequelize 설치

- create express

  `express learn-sequelize --view=pug`

- install package

  ```shell
  cd learn-sequelize/
  npm i
  ```

- install sequelize, mysql2

  ```shell
  npm i sequelize mysql2
  npm i -g sequelize-cli //명령어 설치
  ```

- initiate sequelize

  `sequelize init`
